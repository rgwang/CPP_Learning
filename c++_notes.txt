1.左值与右值
左值是指可出现在赋值操作符“=”左边的各种变量，右值是指可出现在赋值操作符右边的各种可求值的表达式。

2.若变量未初始化，实际上这种变量仍有某个毫无意义的值，其取决于0的形式和最近一次使用这部分内存的程序所留下来的值，因此最好在声明变量的同时就给变量赋值。
       --类名 变量名 = 表达式，如  int count = 0;
       --类名 变量名(表达式)，如 int count(0);

3.赋值操作符(=)与算术操作符写在一起。如 count += 2 等同 count = count + 2.

4.定义函数必须指定4个元素：返回类型、函数名、圆括号内的形参表（可能为空）和函数体。

5.标准库iostream定义了4个IO对象。标准输入cin(istream类)，标准输出cout(ostream类)，还有2个ostream对象cerr和clog，分别用来输出警告、错误信息和产生程序执行的一般信息。
